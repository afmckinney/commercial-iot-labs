Edge Device Management

Part 1: Setup a MQTT and MongoDB Server

Setup the Mosquitto MQTT Broker

On the gateway type the following:

$ adduser mosquitto
$ mosquitto

This will create a user for your Mosquitto and launch the MQTT Broker named mosquitto.

Set up MongoDB on the Intel Iot Gateway

  1. SSH into your Intel Iot Gateway and login.

    If you need a reminder, look at Step 1: Login to your Intel Iot Gateway from the Intro Module.

  2. Download the MongoDB software.

    $ wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-3.0.6.tgz

  3. Once downloaded, unarchive the downloaded file.

    $ tar zxvf mongodb-linux-x86_64-3.0.6.tgz

  4. Add the mongodb binaries to the /usr/bin directory

    $ mv ./bin/* /usr/bin/

  5. The Intel Iot Gateway is enabled with grsecurity which by default prevents the execution of processes that are not added to a white list.

    Add the MongoDM daemon (mongod) to the whitelist type:

    $ paxctl -Cm /usr/bin/mongod

  6. The directory /data/db is the default directory that MongoDB stores its data. Let's make sure that it has plenty of space.

    Type df -h and notice the 28GB partition on /dev/sda3

    Create a directory named /data by typing

    $ mkdir /data

    Then create a link from /data/db to /mnt/space

    $ ln -s /mnt/space /data/db

  7. Start the MongoDM daemon using the "mongod" command:

    $ mongod
