Edge Device Management

Part 1: Provisioning Sensors and Actuators on a Managed Iot Network

Objective

This lab is divided into four parts.

  1. Setup a MQTT Server and a Mongo database.
  2. Write MongooseJS schema and data models.
  3. Write a NodeJS server than monitors the temperature and publishes it over MQTT
  4. Write a NodeJS server than receives Restful HTTP requests to control an edge device
