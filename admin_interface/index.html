Administration Interface

You can monitor the status of your edge network system using NodeJS on the back end and HTML, CSS, and JavaScript on the front end. Write system condition based monitoring triggers, record system errors, and perform edge network analytics.

Objective

By the end of this module, you should be able to:

  • A web-based administrative interface deployed on your Intel IoT Gateway.
  • Have an understanding of how you could build your own web-based monitoring system.

Install the supplied administrative interface

  1. On your development computer’s Desktop, locate gateway-frontend.zip.

  2. Use FileZilla to transfer the zip to your Intel IoT Gateway.

  3. Use PuTTY to SSH into the Intel Iot Gateway.

  4. Navigate to the directory with the zip file and unzip it.

  5. After the zip file has been uncompressed, go into the resulting folder.

    cd gateway-frontend

  6. Run server.js as a Node process.

    node server.js

    This will start a NodeJS HTTP server that will provide the front end user interface.

  7. Visit http://.... in your development computer's web browser.

  8. Explore the interface for condition based monitoring triggers, record system errors, and perform edge network analytics. Refer to the files in gateway-frontend for how these features were implemented.
